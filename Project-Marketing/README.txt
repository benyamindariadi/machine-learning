Folder ini terdiri dari beberapa file dan folder, yaitu:

- bank-additional-full.csv : dataset yang digunakan pada pekerjaan ini.
- clean_df.csv : dataset yang sudah selesai dilakukan tahap preprocessing.
- Data analysis & Modeling.ipynb : Berisi pemrosesan data dari data preparation, data analysis dan data modeling.
- Ready Model for Deployment (Decision Tree).ipynb : Berisi pemodelan final yang akan digunakan untuk deployment.
- Penjelasan.pdf : Berisi penjelasan tentang latar belakang kasus, tahapan pengerjaan dan pembahasan tentang hasil akhir model terhadap latar belakang dan tujuan pemodelan. Berisi juga sedikit penjelasan tentang deployment.

Folder deployment terdiri dari beberapa file, yaitu:
- app.py : Rancangan aplikasi dengan flask untuk deployment.
- model.pkl : Hasil akhir model yang siap dideploy.
- scaler.pkl : Model untuk normalisasi input yang sudah dikalibrasi.
- Folder templates: Berisi 2 file HTML (home.html dan prediction.html) untuk tampilan front end aplikasi yang dideploy.


Hasil deployment berupa online app yang dapat dilihat di: https://bank-marketing-app.herokuapp.com/
